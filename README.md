# admin-mall

## 介绍

一款在线商城的后台管理系统项目。后端采用 Egg + MySQL ，前端采用 Vue + Element 。

## 启动

启动 egg ：

```shell
npm run dev
```

启动 vue：

```shell
cd client
npm run serve
```

## 技术栈

### 前端

- Vue全家桶
- Element-UI

### 后端

- EggJS
- MySQL
